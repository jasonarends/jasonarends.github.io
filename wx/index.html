I participate in SkyWarn weather spotting when I can and have setup an image to be published from my local software to this page, so anyone interested can see what in particular I'm looking at.

It appears the export follows my zoom and pan so if you refresh, you'll see what I'm following.  The first image is the base reflectivity at 0.5 degrees and second is base velocity at 0.5 degrees.  It's a few minutes behind so don't rely on this for emergency use.
